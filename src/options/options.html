設定ガイド & ドキュメント

拡張機能単体でもパワフルに動作

拡張機能だけでも「ローカル保管 + 自動入力 + 診断」までフル活用できます。
必要に応じて、有料プランでクラウド同期（複数端末連携）も追加可能です。

ローカル暗号化 (AES-256-GCM) リスクベース自動入力 フォーム学習 移行プレビュー

拡張機能だけでも使える？

使えます。拡張機能は単体で完結します（ローカルVault運用）。ただし「複数端末の同期」だけは、サーバー（有料）を前提にしています。

  • 拡張機能だけでできる: 保存 / 自動入力 / バックアップ / 移行 / セキュリティ診断
  • サーバーが必要: 課金 / クラウド同期 / 緊急アクセス（暗号化データ回収）

この拡張機能でできること

  • ローカル暗号化されたVaultに、ログイン / カード / 個人情報 / ノートを安全に保存
  • 現在開いているアクティブサイト向けの候補を自動表示してワンクリック自動入力
  • リスクベース自動入力（怪しいドメインやHTTPSではない場合は警告またはブロック）
  • サイト別フォーム学習（自動入力の失敗を手動修正すると学習し、次回から成功率が上がります。誤学習のリセットも可能）
  • フォーム送信内容から新規ログインを検出し、保存候補としてポップアップで提案
  • ローカルセキュリティ診断 + 改善優先度（使い回し、弱いパスワードなど）つきコーチ付き
  • 他サービスからの移行（1Password / Bitwarden / LastPass / 汎用CSV/JSON対応）と差分プレビュー機能
  • 暗号化JSONバックアップによる手動エクスポート / インポート
  • クラウド同期（有料）: サーバーに最新の暗号化状態をスナップショットとして保存し、シームレスに複数端末と同期

使い方（最短スタートガイド）

  1. 1
    Vaultを作成
    ツールバーの拡張機能アイコンをクリックし、マスターパスワードを設定してVault（金庫）を作成します。
  2. 2
    ログイン項目を保存
    「+ アイテム追加」からログイン情報（URL、ユーザー、パスワード）を登録します。
  3. 3
    ページで自動入力
    保存したURLのログインページを開き、拡張機能の「このサイト向け候補」から自動入力を実行します。

⚠️ 注意点（必ずお読みください）

  • マスターパスワードの紛失: 忘れるとデータを復元・復号することは一切できません。必ず安全な場所に記憶・記録してください。
  • クラウド同期の制限: 「有料ユーザーのみ」利用可能です。サーバー側でStripeの利用権ステータスを毎回検証します。
  • 自動入力の精度: サイトごとのDOM（HTML構造）差分により、自動入力がうまく機能しないことがあります。
  • フォーム学習の限界: 自動学習が誤る場合があります。違和感があれば設定から「リセット」を行ってください。

フォーム学習のリセット

ここでは、拡張機能が覚えた自動入力の学習結果を確認して、不要なものだけ削除できます。 「設定からリセットできる」と案内している機能を、実際にこの画面から使えるようにしています。

拡張機能の Options 画面として開くと、学習一覧を表示できます。

学習データはまだありません。自動入力を手動修正してログインすると、ここに履歴が出ます。

    Q. 候補が出ない（自動入力タブが空になる）

    保存したログイン項目に「URL」が正しく入力されているか確認してください。
    （例: https://example.com/login）

    Q. 自動入力先のフィールドがずれる

    一度「手動」で正しいフィールドに情報をコピー＆ペースト入力してログインしてください。パスワードマネージャーがそれを学習し、次回から修正されます。

    Q. 学習が壊れて何度やっても失敗する

    拡張機能設定から「このサイトの学習をリセット」または「全サイトの学習をリセット」を実行してください。

    Q. クラウド同期ができない / エラーになる

    有料プランが有効か、同期ベースURLが正しいか、クラウドタブで「ログイン完了」状態になっているかを確認してください。
