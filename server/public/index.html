PasswordManeger Cloud Layer

モバイルの質感で統一した クラウド運用ハブ

ローカルVaultの安全性はそのままに、同期、契約管理、緊急復旧だけを上に積む補助レイヤーです。 Desktop やモバイルと見た目の温度差が出ないよう、同じダークトーンとアクセントでまとめています。

ここで扱うのは平文ではなく、あくまで暗号化済みスナップショットと利用権です。 つまり「保管庫本体」ではなく、「安全に運ぶための導線」を管理する画面です。

Cloud Sync 暗号化Vaultを push / pull

有料ユーザーだけが複数端末の保管庫を同期できます。

Billing Stripe と同じ導線で契約管理

Checkout と Portal をひとつの画面から扱えます。

Recovery 緊急復旧の準備を先回り

鍵分割と暗号化データ取得を事故対応導線として残せます。

Cloud Onboarding

新規登録

まずはクラウドアカウントを作成します。支払い前でも登録だけは先にできます。

同期を解放 プレミアム化すると Desktop / 拡張 / モバイルで同じVaultを扱えます。
復旧も準備 課金導線と緊急アクセス導線を分離せず、この画面で確認できます。

Existing Account

ログイン

契約状態や緊急アクセスはログイン後に同じ画面から操作できます。

Premium Access

課金

支払い対象はローカル保管庫そのものではなく、暗号化同期、契約管理、緊急復旧導線です。 つまり「運用面の安全性と継続性」に対する課金です。

同期を有効化

Desktop、拡張機能、モバイル間で暗号化Vaultを安全に push / pull できます。

契約を一元管理

Checkout と Billing Portal を同じブランド体験のまま往復できます。

復旧準備を残す

鍵分割や緊急アクセス用の暗号化データ取得導線を、支払い後も整理して扱えます。

ログイン前は課金開始・契約管理・緊急アクセスを操作できません。まず認証タブで接続してください。
