Autofill Playground

PasswordManeger Demo Pages

ブラウザ拡張の「自動入力」「保存候補検出」「サイト別フォーム学習」を安全に試すためのデモページです。 ここはローカル開発用で、実際の外部サイトにはアクセスしません。

使い方（最短）

1. サーバー起動: npm run server:dev
2. 拡張機能を読み込み（Chrome: chrome://extensions）
3. 拡張機能のポップアップでVault作成→ログイン項目を作成（URLは http://localhost:8787/demo/login.html など）
4. このページから各フォームへ移動し、ポップアップの候補から自動入力

Login フォーム

標準フォームと、少しクセがあるフォームを切り替えられます。
Trickyフォームでは id/name を変えて、フォーム学習の効き方も確認できます。

Card フォーム

カード番号/名義/期限/CVC を入力するフォームです。

Identity フォーム

氏名/メール/電話/住所 の入力フォームです。

リスクベース自動入力の試し方

リスク判定は「保存したURLのドメイン」と「今見ているページのドメイン」が一致するか、HTTPSか、などで決まります。
たとえば「URLが https://example.com のログイン項目」を作って、このデモページ（localhost）で自動入力すると、 高リスクになりやすいです（ブロック or 確認）。
