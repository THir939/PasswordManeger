Learning Login Form

自動入力が最初は外れやすいフォームを使い、「手動修正→送信」で学習が効くかを確認するページです。 通信はしません。

使い方

1. 拡張でこのURL向けログイン項目を作成
2. 「このサイトに入力」を押す（最初は囮フィールドに入る）
3. 本来のユーザー名欄に手動修正して送信
4. もう一度「このサイトに入力」で本来欄に入るか確認

Learning Target Form

先頭の「メモ欄（囮）」があるため、未学習時はここに入ってしまうことがあります。
本来欄は placeholder + aria-label を固定し、DOM変化時の学習適用を確認しやすくしています。
